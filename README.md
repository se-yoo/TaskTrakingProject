# TaskTrakingProject
### 업무 문의 조회 시스템 ###

- 업무 문의를 접수 받고 진행 현황을 관리하는 윈도우폼 시스템
<div>
  <img src="https://img.shields.io/badge/language-C%23-239120?style=flat-square" />
</div>

<br>

## 기능 ##
### 관리자 ###
- 진행별 문의 조회(진행중부터는 자신이 담당인 문의만 조회)
- 문의 담당
- 문의 처리 이력 작성

### 사용자 ###
- 문의 등록/수정/삭제
- 문의 변경사항 알림(담당자 배정, 진행코드변경)
